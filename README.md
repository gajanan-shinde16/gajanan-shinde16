## 🚀 Featured Projects

### 🎯 [Offerly](https://github.com/gajanan-shinde16/offerly) — Placement Application Tracker

> Full-stack web app to track, manage, and analyze campus placement applications

- **Stack:** React (Vite) · Tailwind CSS · Node.js · Express · MongoDB · JWT · Zod · Recharts
- Role-based access control — separate Student & Admin dashboards
- JWT stored in **HttpOnly cookies** (security-first, not localStorage)
- Analytics — offers vs rejections, company-wise stats, round-wise drop-off
- Input validation with **Zod** · Deployed on **Vercel**
